category: Components
title: Toast
---

用来弹出一定时间内可以自动消失的提示

## API

| 参数        | 说明                                     | 类型        | 默认值 |
|----------- |----------------------------------------- | ----------  | ------- |
| msg       | 弹出来的文案                          | String     | `toast`  |
| duration  | toast持续的时间                       | Number       | `1000(ms)` |
